    width="100%"
    height="400"
    src="PASTE-YOUR-EMBED-URL-HERE"
    frameborder="0"
    allowfullscreen>
  </iframe>
</div>

<!--
  HOW TO GET EMBED URLS:
  Ultiplays: Share button → Copy Embed → use the src="..." URL only
  YouTube:   Share → Embed → copy the src="..." URL (looks like youtube.com/embed/VIDEOID)
  Vimeo:     Share → Embed → copy the src="..." URL
-->
Tip: Only paste the URL that goes inside src="..." — not the full embed code block. Change the height number to make the player taller or shorter.

Collapsible Section

Inside Joel's Mind

▾

This content is hidden until the user clicks to expand it. Great for optional reading, long notes, or anything that shouldn't clutter the main page.

You can put any content in here — text, bullet lists, images, even more code blocks.

<!-- Collapsible / Expandable Section -->
<!-- Clicking the summary row toggles the content open/closed -->

<details class="collapsible">

  <!-- This is the clickable header bar -->
  <summary>
    <h3>Inside Joel's Mind</h3>
    <span class="chevron">▾</span>
  </summary>

  <!-- This content shows when expanded -->
  <div class="coll-body">
    <p>Your hidden content goes here. Can be as long as you like.</p>
    <p>Add more paragraphs, lists, images — anything.</p>
  </div>

</details>
Tip: Add open to the <details> tag (e.g. <details class="collapsible" open>) if you want it to start expanded by default.
