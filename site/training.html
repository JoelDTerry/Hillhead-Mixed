How to add a new session plan

  1. Duplicate the file session-12-april.html and rename it, e.g. session-19-april.html
  2. Open the new file and update all the session content inside
  3. Open training.html and copy the plan card block, updating the date, title, tags and href
  4. Save both files and drag the whole site folder to Netlify to redeploy
