How to add a scouting report

  1. Create a new HTML file named after the opponent, e.g. scout-team-name.html — copy session-12-april.html as a starting template.
  2. Replace the session content with your scouting notes: offensive tendencies, defensive looks, key players to watch, play diagrams.
  3. Open scouting.html and add a card linking to your new file — follow the same pattern as the plan cards in training.html.
  4. If you have footage of the opponent, paste the video embed into their scouting page at the bottom.
  5. Save all files and re-upload the site folder to Netlify.
