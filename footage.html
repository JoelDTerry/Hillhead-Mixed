How to add footage

  1. Get an embed link from your video platform — YouTube, Vimeo, or Ultiplays all provide embed codes via a Share → Embed option.
  2. Open footage.html in VS Code.
  3. Remove or hide the empty-state block (the grey dashed box section).
  4. Add a card for each video using a <div class="footage-card"> block — paste the iframe embed inside it, then add a title and date below.
  5. Save the file and re-upload the site folder to Netlify.
  6. Repeat for each new game or session recording.
